S3-хранилище

Масштабируемое объектное хранилище для любых типов данных. Идеально подходит для хранения контента сайтов, резервных копий, логов и аналитических данных. Полная совместимость с S3 API — стандартом для объектных хранилищ.

Преимущества:

  • ✓ Безлимитное масштабирование
  • ✓ Надёжность 99.99%
  • ✓ Интеграция с CDN
  • ✓ Гибкое управление доступом
  • ✓ Версионирование файлов
  • ✓ Шифрование данных

Варианты использования:

  • Хостинг статических сайтов
  • Хранение медиа-контента
  • Резервное копирование
  • Аналитические данные

BaaS (Backup as a Service)

Автоматическое резервное копирование ваших данных с гибкими настройками расписания и глубины хранения. Защита от вирусов-шифровальщиков.

Ключевые возможности:

  • ✓ Резервное копирование 24/7
  • ✓ Мгновенное восстановление
  • ✓ Хранение в нескольких ЦОД
  • ✓ Дедупликация данных
  • ✓ Инкрементальные копии
  • ✓ Политики хранения до 7 лет

Поддерживаемые системы:

  • Windows Server, Linux
  • VMware, Hyper-V
  • Microsoft 365, Exchange
  • Базы данных: MySQL, PostgreSQL, MSSQL

DRaaS (Disaster Recovery as a Service)

Аварийное восстановление критически важных систем. Гарантированное время восстановления (RTO) и точка восстановления (RPO) в соответствии с вашими потребностями.

SLA гарантии:

  • ✓ RTO от 15 минут
  • ✓ RPO от 5 минут
  • ✓ Регулярное тестирование
  • ✓ Автоматическое переключение
  • ✓ Географическая распределённость

Тарифные планы (цена за ГБ):

Тариф RTO RPO Цена за ГБ
Базовый 4 часа 24 часа 0.30 BYN
Бизнес 1 час 1 час 0.40 BYN
Критический 15 мин 5 мин 0.60 BYN

Ресурс Цена

SSD диск 0.30 BYN/ГБ в месяц
HDD диск 0.15 BYN/ГБ в месяц
